# 智谱清言创图 - UI 资源提示词手册

**创建日期**: 2026-04-17  
**项目**: 封狼居胥 - 自走棋微信小程序  
**AI 工具**: 智谱清言 - 创图功能  
**用途**: 批量生成游戏 UI 资源，包含详细提示词、文件名和保存路径

---

## 📋 使用说明

1. **复制提示词**: 将完整提示词直接复制到智谱清言创图输入框
2. **生成图片**: 点击"生成"按钮
3. **下载保存**: 保存为指定文件名，放到对应路径
4. **导入 Cocos**: 在 Cocos Creator 中刷新资源

---

## ⚠️ 透明背景说明

所有提示词已包含**纯透明背景**要求。如果仍有背景色：
- 使用 https://www.remove.bg/ 自动抠图
- 或在 Photoshop 中删除背景后导出 PNG
