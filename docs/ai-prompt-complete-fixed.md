## 关键透明背景关键词说明

### Midjourney/Stable Diffusion 不支持真正透明

⚠️ **重要提示**: Midjourney 和 SD 生成的图片**不是真正的透明 PNG**,而是白色/灰色背景。

### 实际工作流程

1. **AI 生成** → 白色/灰色背景
2. **后期抠图** → Photoshop/在线工具去除背景
3. **保存 PNG** → 带透明通道的 PNG 文件

### 推荐的抠图工具

```
1. Photoshop: 
   - 魔棒工具选白色 → 删除 → 保存 PNG
   
2. 在线工具:
   - remove.bg (自动抠图)
   - clippingmagic.com
   - photopea.com (免费在线 PS)
   
3. 批量处理:
   - 使用 Photoshop Action 批量抠图
```

### 提示词中的"透明背景关键词"

虽然 AI 不能直接生成透明图，但这些关键词有帮助:

```
✅ transparent background
✅ checkerboard pattern  
✅ alpha channel
✅ isolated on transparent
✅ clean edges for cutout
✅ no background
```

这些词会让 AI 生成**更容易抠图**的图片 (边缘清晰、主体明确)
