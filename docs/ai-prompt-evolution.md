## 质量检查

### 转职形态检查清单

```
视觉区分:
□ 体型是否明显更大 (1.3x/1.5x)?
□ 装备是否更华丽 (细节更多)?
□ 特效是否更强力 (粒子更多)?
□ 配色是否更高级 (金色/传奇色)?

风格统一:
□ 是否保持手绘涂鸦风格?
□ 线条粗细是否一致?
□ 色彩饱和度是否统一?
□ 光影方向是否一致?

技术规格:
□ 透明背景是否正确?
□ 边缘是否干净适合抠图?
□ 尺寸是否符合@2x/@3x?
□ 无水印无文字?

年龄适宜:
□ 是否保持 Q 版可爱?
□ 是否避免恐怖血腥?
□ 是否适合 12+ 岁?
```
