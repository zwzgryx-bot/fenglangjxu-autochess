## 使用提示词时的注意事项

1. **汉军必须包含**: `vermilion red and dark green` (朱砂红 + 墨绿)
2. **匈奴必须包含**: `earth yellow and brown` (土黄 + 棕色)
3. **避免颜色混淆**: 不要让汉军用土黄色，匈奴用墨绿色
4. **金色点缀**: 5 费传说棋子可以加金色装饰
5. **品质区分**: 低费棋子颜色简单，高费棋子颜色华丽

## 测试流程

生成第一批棋子时的验证顺序:

1. **生成霍去病** - 验证朱砂红 + 墨绿配色
2. **生成单于** - 验证土黄 + 棕色配色
3. **放在一起对比** - 确保两个阵营色彩区分明显
4. **调整提示词** - 如果颜色不对，调整色彩描述的权重
5. **批量生成** - 确认配色正确后批量生产
