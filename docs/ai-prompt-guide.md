## Stable Diffusion 专用参数

### Recommended Settings
```
Model: Anything V5 / Counterfeit V3 (动漫卡通风格较好)
Sampler: DPM++ 2M Karras
Steps: 28-35
CFG Scale: 7
Size: 512x768 (竖图), 768x512 (横图)
Hires. fix: 启用，2x 放大
```

### ControlNet 配置
```
使用 ControlNet 保持角色一致性:
- 启用 Canny 边缘检测
- 启用 Open Pose 控制姿势
- 启用 Depth 控制深度
```

## 后期处理流程

### Photoshop 处理步骤

1. **抠图**
   - 使用魔棒/钢笔工具抠出主体
   - 删除背景，保存为透明 PNG

2. **统一色调**
   - 添加色彩平衡调整层
   - 统一暖色调 (羊皮纸风格)

3. **强化线条**
   - 复制图层 → 滤镜 → 查找边缘
   - 混合模式：正片叠底
   - 增强手绘线条感

4. **添加纹理**
   - 叠加羊皮纸纹理素材
   - 混合模式：柔光，透明度 20%

5. **输出切图**
   - @2x: 原尺寸导出
   - @3x: 150% 放大导出
   - 格式：PNG-24 (透明)

## 批量生成技巧

### Midjourney 参数优化
```
--ar 宽高比 (3:4 立绘，16:9 场景，1:1 图标)
--v 5.2 使用最新版本
--stylize 250 风格化程度 (0-1000)
--chaos 20 随机性 (0-100)
--seed 1234 固定随机种子保持风格
```

### 保持一致性的方法

1. **固定 Seed**: 使用相同 seed 值生成同系列
2. **垫图功能**: 用第一张满意图作为后续生成的参考
3. **风格词固定**: 保持核心风格词不变
4. **色板限定**: 指定具体的色彩代码

## 推荐工作流

```
1. Midjourney 生成初稿 → 2. 筛选满意图 → 
3. Photoshop 抠图修图 → 4. 统一调色 → 
5. 输出多倍率切图 → 6. 导入 Cocos Creator
```

**时间估算**:
- AI 生成：30 分钟/张 (含调整)
- 后期处理：15 分钟/张
- 80 棋子立绘：约 60 工时

**建议**: 优先制作 10 个核心武将确认风格，再批量生产!
