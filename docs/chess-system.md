# 封狼居胥 - 兵种与升级系统详解

## 兵种 (职业) 体系

### 双阵营 × 5 职业架构

游戏采用 **双阵营 × 5 职业** 的兵种架构，共计 80+ 棋子。

```
┌─────────────────────────────────────┐
│         封狼居胥兵种体系            │
├─────────────────────────────────────┤
│  汉军阵营 (45 棋子)    匈奴阵营 (40 棋子) │
│  ├─ 骑兵 (15)         ├─ 狼骑兵 (15)   │
│  ├─ 弩兵 (12)         ├─ 弓骑兵 (12)   │
│  ├─ 车兵 (8)          ├─ 萨满 (8)      │
│  ├─ 谋士 (5)          ├─ 勇士 (5)      │
│  └─ 武将 (5)          └─ (可有补充)    │
└─────────────────────────────────────┘
```

## 棋子升级系统

### 升星系统 (3 合 1)

```
1 星 → 2 星：3 个相同 1 星棋子 = 1 个 2 星棋子
2 星 → 3 星：3 个相同 2 星棋子 = 1 个 3 星棋子

属性提升倍率:
- 2 星：基础属性 × 1.5
- 3 星：基础属性 × 2.5
```

**示例**:
```
霍去病 (1 星): HP 800, ATK 120, DEF 40
霍去病 (2 星): HP 1200, ATK 180, DEF 60
霍去病 (3 星): HP 2000, ATK 300, DEF 100
```

### 玩家等级系统 (人口升级)

**等级影响**:
- 可上阵棋子数量
- 商店刷新棋子品质概率
- 可容纳的棋子总数

**升级费用和人口**:

| 4 | 6 | 8 | 12 |
| 5 | 7 | 16 | 28 |
| 6 | 8 | 24 | 52 |
| 7 | 9 | 32 | 84 |
| 8 | 10 | 40 | 124 |

**商店概率随等级变化**:

### 装备系统 (可选扩展)

**如果后期加入装备系统**:

装备类型:
1. **武器** - 增加攻击力、攻速、暴击
2. **防具** - 增加生命值、防御力、抗性
3. **饰品** - 特殊效果 (吸血、闪避、能量获取)

装备获取:
- 野怪回合掉落
- 商店随机刷新
- 任务奖励

装备合成:
```
3 个 1 级装备 = 1 个 2 级装备
3 个 2 级装备 = 1 个 3 级装备
```

## 使用建议

### 新手推荐阵容

**汉军骑兵流**:
```
核心：霍去病 + 卫青
搭配：公孙贺 + 赵破奴 + 其他骑兵
羁绊：4 骑兵 + 3 汉军
```

**匈奴狼骑流**:
```
核心：单于 + 左贤王 + 右贤王
搭配：狼骑兵 ×3
羁绊：6 狼骑兵 + 3 匈奴
```

**混合羁绊流**:
```
核心：李广 (弩兵) + 霍去病 (骑兵)
搭配：车兵坦克 + 谋士辅助
羁绊：2 弩兵 + 2 骑兵 + 2 车兵
```

### 高级策略

- **经济运营**: 前期攒钱吃利息，中期升人口找核心
- **阵容转型**: 根据发牌员灵活切换阵营
- **装备分配**: 优先给核心输出位
- **站位调整**: 根据对手调整坦克/输出位置
