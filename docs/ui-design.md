# 封狼居胥自走棋 - UI 设计文档

## 整体风格定义

### 美术风格：手绘涂鸦卡通

**关键词**: 粗犷线条、手绘质感、温暖色彩、轻松有趣

### 色彩规范

```css
/* 主色系 */
--bg-parchment: #F5E6C8;      /* 羊皮纸背景 */
--bg-dark: #E8D5A7;           /* 深色背景 */
--border-charcoal: #2C2C2C;   /* 炭笔边框 */

/* 功能色 */
--primary-red: #C43628;       /* 朱砂红 - 主按钮 */
--primary-green: #3A5F3A;     /* 墨绿 - 汉军 */
--primary-yellow: #B8956B;    /* 土黄 - 匈奴 */
--accent-gold: #D4AF37;       /* 金色 - 传说 */

/* 品质色 */
--quality-common: #B0B0B0;    /* 白色/普通 */
--quality-uncommon: #68C257;  /* 绿色/稀有 */
--quality-rare: #4A90DB;      /* 蓝色/史诗 */
--quality-epic: #A368E8;      /* 紫色/传说 */
--quality-legendary: #F0A02D; /* 橙色/神话 */

/* 状态色 */
--success: #5CB85C;
--warning: #F0AD4E;
--danger: #D9534F;
--info: #5BC0DE;
```

### 线条风格
- **边框粗细**: 3-5px 手绘抖动画笔
- **圆角**: 8-12px 不等圆角 (故意不完全对称)
- **阴影**: 手绘排线阴影，非渐变

## 美术资源制作工具推荐

### 手绘风格实现方案

**方案 A: 纯手绘 (推荐)**
- 工具：Procreate / Photoshop + 手绘板
- 流程: 线稿 → 上色 → 导出透明 PNG
- 优点：质感最好，独特性强
- 缺点：成本高，周期长

**方案 B: AI 生成 + 手绘修饰**
- 工具: Midjourney / Stable Diffusion + Photoshop
- 流程: AI 生成 → 人工修图 → 统一风格
- 优点：速度快，成本低
- 缺点：风格统一性需人工把控

**方案 C: 3D 渲染卡通**
- 工具：Blender + 卡通着色器
- 流程: 3D 建模 → 卡通渲染 → 输出 2D
- 优点: 可复用，动作流畅
- 缺点：技术门槛高
