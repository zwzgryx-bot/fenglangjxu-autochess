# 封狼居胥 - 兵种转职系统

## 设计理念

**核心概念**: 低费棋子升到 3 星后，可以转职成更强大的特殊兵种，获得全新技能和定位。

**设计目的**:
- ✅ 增加低费棋子的后期价值
- ✅ 提供"小卡逆袭"的爽快感
- ✅ 丰富阵容搭配策略
- ✅ 鼓励玩家培养"本命英雄"

---

## 转职系统规则

### 基础规则

```
转职条件:
1. 棋子必须达到 3 星⭐⭐⭐
2. 玩家等级达到 5 级及以上
3. 消耗转职道具 (或金币)

转职效果:
- 基础属性提升 30-50%
- 技能完全改变 (更强)
- 外观模型变化 (更大/更炫酷)
- 羁绊不变 (仍算原职业/阵营)

转职不可逆:
- 一旦转职，无法变回原职业
- 转职后不能继续升星 (已是终点)
```

## 转职时机策略

### 何时转职？

**前期转职 (10-15 回合)**:
```
适合: 1 费 3 星棋子
优点: 中期战力强，连胜容易
缺点: 占用经济，可能后期乏力
建议: 只转核心卡，别全转
```

**中期转职 (16-20 回合)**:
```
适合: 2-3 费 3 星棋子
优点: 战力+ 经济平衡
缺点: 可能错过最佳 timing
建议: 最稳妥的选择
```

**后期转职 (21+ 回合)**:
```
适合: 所有 3 星棋子
优点: 经济充裕，随意花
缺点: 可能来不及用就结束
建议: 有资源就转，别省
```

## 平衡性考虑

### 限制机制

```
1. 转职道具限量
   - 每局游戏商店共刷新 5-8 本
   - 野怪掉落固定数量
   - 避免人人全员转职

2. 转职人口限制
   - 每场战斗最多 3 个转职棋子
   - 或：转职棋子占用 2 人口
   - 平衡强度

3. 转职不可逆
   - 防止反复横跳
   - 要求玩家慎重选择
```

### 强度控制

```
转职后强度 ≈ 同星级 4 费卡
终极形态强度 ≈ 5 费卡
但不能超过 5 费原卡
保持稀有度价值
```
