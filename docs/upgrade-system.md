# 封狼居胥 - 兵种升级路线详解

## 升级系统总览

```
┌─────────────────────────────────────────┐
│           兵种升级三大方向              │
├─────────────────────────────────────────┤
│  1. 棋子升星 (1 星→2 星→3 星)            │
│  2. 玩家等级 (1 级→8 级，提升人口)       │
│  3. 羁绊激活 (数量阈值触发)             │
└─────────────────────────────────────────┘
```

---

## 一、棋子升星系统

### 升星规则

```
基础公式:
- 2 星 = 3 个相同 1 星棋子合成
- 3 星 = 3 个相同 2 星棋子合成

属性倍率:
- 1 星：基础值 (100%)
- 2 星：基础值 × 1.5
- 3 星：基础值 × 2.5

技能升级:
- 1 星：基础技能效果
- 2 星：技能效果 +30% 或 CD -20%
- 3 星：技能效果 +60% 或 CD -40% 或 新增效果
```

## 三、玩家等级升级路线

### 等级提升路线图

```
┌──────┬──────┬────────┬──────────┬─────────────────┐
│ 等级 │ 人口 │ 升级费用 │ 累计金币 │ 解锁内容        │
├──────┼──────┼────────┼──────────┼─────────────────┤
│  1   │  3   │   -    │    -     │ 基础商店        │
│  2   │  4   │   -    │    -     │ -               │
│  3   │  5   │   4    │    4     │ 3 费棋子刷新    │
│  4   │  6   │   8    │   12     │ -               │
│  5   │  7   │  16    │   28     │ 4 费棋子刷新    │
│  6   │  8   │  24    │   52     │ -               │
│  7   │  9   │  32    │   84     │ 5 费棋子刷新    │
│  8   │ 10   │  40    │  124     │ 高费概率提升    │
└──────┴──────┴────────┴──────────┴─────────────────┘
```

### 每级解锁的商店概率

```
等级 1: [100/0/0/0/0] - 只刷新 1 费
等级 3: [75/20/5/0/0]  - 解锁 3 费
等级 5: [45/35/15/5/0] - 解锁 4 费
等级 7: [25/30/30/10/5] - 解锁 5 费
等级 8: [15/25/35/15/10] - 5 费概率翻倍
```

### 升级时机建议

**前期 (1-10 回合)**:
```
- 保持 3-4 级，攒钱吃利息
- 只买关键核心卡
- 不上头 D 牌
```

**中期 (11-20 回合)**:
```
- 升到 5-6 级，提升战力
- 适度 D 牌找核心 2 星
- 保持经济 50+ 金币
```

**后期 (21+ 回合)**:
```
- 升到 7-8 级，找 5 费卡
- 该花就花，钱花光也要提升战力
- 3 星核心卡决定胜负
```

## 七、升级资源管理

### 金币分配原则

```
经济阈值:
- 每多 10 金币，额外 +1 利息 (上限 5)
- 保持 50 金币吃满利息
- 超出 50 的部分可以花

D 牌预算:
- 找 2 星核心：预留 30-50 金币
- 找 3 星核心：预留 80-100 金币
- 追 3 星 5 费：预留 150-200 金币

升级预算:
- 升到 5 级：28 金币
- 升到 6 级：52 金币
- 升到 7 级：84 金币
- 升到 8 级：124 金币
```

### 棋子储备策略

```
卡片储备上限:
- 场上：最多 10 个 (8 级时)
- 备战区：最多 8 个
- 总计：18 个棋子

储备优先级:
1. 核心输出 (准备追 3 星)
2. 关键坦克 (2 星即可)
3. 功能辅助 (1-2 星)
4. 羁绊 filler (凑数用)
```

## 八、特殊情况处理

### 1. 核心卡被 D 光

**应对方案**:
```
- 立即转型，不要硬玩
- 转向备胎核心
- 例如：霍去病没了，改玩弩兵流
```

### 2. 经济爆炸

**应对方案**:
```
- 卖棋子回血
- 卖掉高费卡，保留低费 3 星
- 连败收菜，等下波爆发
```

### 3. 被多家同行

**应对方案**:
```
- 提前 D 牌，抢核心
- 或者转阵容，避免内战
- 观察对手，后发制人
```
